Se añade display: grid; a div .grid

A continuación se necesita especificar, grid-template-rows

En caso de que haya más divs o "renglones" que los especificados en grid-template-rows, se ajustan de manera automática a 1fr (Implicit grid o rows.)

Grid Multidimensional

Se pueden usar grid-template-rows con grid-template-columns. En éste casos se usa:

    .grid {
      display: grid;
      grid-template-columns: repeat(2, 1fr);
      grid-template-rows: 1fr 2fr 3fr;
    }
